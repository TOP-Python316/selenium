Краткая информация по HTML5 и CSS3

Базовые HTML5 теги

HTML (Hypertext Markup Language) — язык разметки веб-страниц.

  • html — корневой элемент
  • head — информация о документе
  • title — заголовок документа
  • body — содержимое документа
  • h1 — заголовок первого уровня. Бывают от h1 до h6
  • div — блочный элемент, который позволяет группировать другие элементы вместе.
  • p — абзац
  • ul — маркированный список
  • ol — нумерованный список
  • li — элемент списка
  • img — изображение
  • a — ссылка

Семантические теги

Семантические теги позволяют браузерам и поисковым системам понимать структуру веб-страниц.

  1. header — шапка сайта
  2. nav — навигация
  3. main — основное содержимое
  4. section — раздел документа
  5. article — самостоятельная статья
  6. footer — подвал сайта

Картинки и ссылки

Картинки можно добавлять с помощью тега img.

Ссылки можно добавлять с помощью тега a.

Изображение и его атрибуты

Атрибуты src и alt являются обязательными для тега img.

Атрибут src содержит путь к изображению.

Атрибут alt содержит информацию о изображении.

Ссылка и её атрибуты

Атрибут href содержит путь к ссылке и он обязателен для тега a.

title — всплывающая подсказка.

CSS селекторы

Селекторы CSS позволяют выбрать элементы веб-страницы и применять к ним стили.

  • element - выбор элемента
  • .class - выбор элемента по классу
  • element.class - выбор элемента с определенным классом
  • #id - выбор элемента по идентификатору
  • element, element - выбор нескольких элементов
  • element element - выбор вложенных элементов
  • element > element - выбор дочерних элементов
  • element[attribute] - выбор элемента по атрибуту
  • element[attribute="value"] - выбор элемента по атрибуту и его значению
